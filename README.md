# Elasticsearch Cookbook

This project focuses on anomaly detection in a SIEM environment using Machine Learning techniques. The objective is to identify unusual or potentially malicious behaviors within system and network logs by leveraging data analysis and predictive models. The notebook included in this repository demonstrates the full pipeline, from data preprocessing and feature engineering to model training and evaluation.
